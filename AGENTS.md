# AGENTS.md

Guidance for coding agents working in this repository.

## Project context

Crust is a Visual Studio Code extension intended to provide a true VS Code UI for the [Pi Coding Agent](https://pi.dev/). The goal is feature parity with comparable coding-agent extensions, including session browsing, IDE context integration, chat export, and diff viewing.

This project is at a very early stage. The current contents of `src/` are generated by the VS Code extension template and should not be treated as established application architecture.

## Working guidelines

- Treat `README.md` as the source of truth for current project goals and status.
- Do not preserve template-generated code in `src/` merely for compatibility if real implementation work requires replacing it.
- Prefer small, focused changes with clear separation between extension activation, Pi integration, UI/webview code, and VS Code command registration.
- Keep user-facing behavior aligned with VS Code extension conventions.
- Assume Pi must be installed and available on `PATH`, as documented in `README.md`.

## Development commands

Use the scripts defined in `package.json` for validation and builds. Common checks include:

```sh
npm run compile
npm run lint
```

## Notes

- Avoid adding large generated artifacts unless they are required by the extension packaging workflow.
- Keep documentation updated as features move beyond the template-generated starting point.
